==================================================================
- Esrutura condicional " if "

if Condição:
    comando_1
    comando_2

    Condição -> qualquer operação que,
    utilizando operadores lógicos e/ou
    relacionais, resulte em verdaddeiro
    (True) ou falso (False)

===================================================================
- Esrutura condicional " else "

if CONDICAO:
    comando_1
    comando_2
else:
    comando_se_não

comando_se_não sempre será executado
se a CONDICAO não for verdadeira


===================================================================
- Esrutura condicional " elif "

if CONDICAO_1:
    comando_1
    comando_2
elif CONDICAO_2:
    comando_3
    comando_4
else:
    comando_se_não


===================================================================
