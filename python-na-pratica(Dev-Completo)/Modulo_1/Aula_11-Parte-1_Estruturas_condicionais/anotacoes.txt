==================================================================
- Esrutura condicional " if "

if Condição:
    comando_1
    comando_2

    Condição -> qualquer operação que,
    utilizando operadores lógicos e/ou
    relacionais, resulte em verdaddeiro
    (True) ou falso (False)

===================================================================
- Esrutura condicional " else "

if CONDICAO:
    comando_1
    comando_2
else:
    comando_se_não

comando_se_não sempre será executado
se a CONDICAO não for verdadeira


===================================================================
- Esrutura condicional " elif "

if CONDICAO_1:
    comando_1
    comando_2
elif CONDICAO_2:
    comando_3
    comando_4
else:
    comando_se_não


===================================================================


Quando usar cada um?

if (Se)
É sempre o ponto de partida. Use quando você tem uma condição principal que precisa ser testada.


elif (Senão, se)
É o intermediário. Use quando a primeira condição foi falsa, mas você ainda tem outra regra específica para testar antes de desistir. Você pode ter quantos elif precisar.


else (Senão / Caso contrário)
É a "rede de segurança". Ele não aceita nenhuma condição (não se escreve nada na frente dele). Ele serve para capturar tudo o que sobrou.
